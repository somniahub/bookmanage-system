# springboot-vue图书管理系统

## 简介

主要界面有书籍管理、用户管理、权限管理、订单管理等

## 技术栈

Vue、Element-ui、axios、git、JavaScript

## 主要技术实现

基于 Vue、JS 运用 Element-ui 进行页面编写

主页使用 echarts 绘制各地区用户比例图

使用 vue-router 实现前端各个页面之间的管理、跳转
